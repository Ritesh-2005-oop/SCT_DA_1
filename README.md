# Superstore Sales Dashboard

## Project Overview

This project presents an interactive Excel dashboard built using Superstore Sales data. The dashboard provides key business insights into sales performance, profit trends, customer segments, and regional performance.

## Objectives

* Analyze overall sales and profit performance.
* Identify top-performing products and categories.
* Compare sales across different regions and segments.
* Visualize business trends through interactive charts and KPIs.

## Tools Used

* Microsoft Excel

  * Pivot Tables
  * Pivot Charts
  * Slicers
  * Conditional Formatting
  * Data Cleaning and Transformation

## Dashboard Features

* Total Sales Analysis
* Profit Analysis
* Category-wise Performance
* Region-wise Sales Distribution
* Customer Segment Analysis
* Interactive Filters and Slicers

## Files Included

* **Superstore_Sales_Dashboard.xlsx** – Interactive Excel dashboard.
* **README.md** – Project documentation.

## Key Insights

* Identified top-performing product categories.
* Analyzed regional sales and profit contribution.
* Highlighted customer segments driving revenue.
* Visualized sales trends to support business decision-making.
